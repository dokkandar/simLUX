How to use

This panel is built to INSPECTOR_DESIGN.md. Open devtools (F12) and hover any element — computed width/height should match the measurement table (§7): field box 140×24, label column 84, coordinate field 64×24, swatch 13×13, row pitch 32, radius 4.

Toggle the button to outline the boxes.

Fonts: Geist/JetBrains Mono if installed, else the system fallback — glyph widths differ slightly but box sizes are exact.
